# pfsense-config

[English](readme_eng.md) | [Русский](readme_rus.md)

EN: Declaratively configures a pfSense router/firewall to a desired state —
system, interfaces, gateways & gateway groups, aliases, firewall rules, DHCP,
SNMP, Unbound, GUI certificate and the admin account. Hybrid engine: native
`pfsensible.core` modules where available, idempotent PHP playback for the rest.

RU: Декларативно настраивает роутер/файрвол pfSense до желаемого состояния —
система, интерфейсы, шлюзы и группы шлюзов, алиасы, правила файрвола, DHCP,
SNMP, Unbound, GUI-сертификат и учётка admin. Гибрид: нативные модули
`pfsensible.core` где есть, идемпотентный PHP playback для остального.
